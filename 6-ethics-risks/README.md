## Topic 6 — Risks, Ethics, and Limitations of AI (topic-6-risks,-ethics,-and-limitations-of-ai)

Students will investigate possible risks and ethical concerns of AI-assisted software development.

Possible areas:
* hallucinations,  
* incorrect code generation,  
* security vulnerabilities,  
* privacy concerns,  
* overdependence on AI,  
* skill degradation,  
* copyright issues,  
* ethical responsibility.

Main question:
* What happens if engineers stop understanding the code they ship?
