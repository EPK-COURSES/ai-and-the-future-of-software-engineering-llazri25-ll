## Topic 3 — Career Strategy and Continuous Learning (topic-3-career-strategy-and-continuous-learning)

Students will investigate how software engineers continue developing themselves in a rapidly changing industry.

Possible areas:
* certificates vs practical experience,  
* portfolio development,  
* open-source participation,  
* self-learning strategies,  
* balancing fundamentals and new technologies,  
* time management for continuous learning.

Main question:
* How can professionals continue adapting when technology changes faster than available time?
