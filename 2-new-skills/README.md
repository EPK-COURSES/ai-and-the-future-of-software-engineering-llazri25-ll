## Topic 2 — New Skills Required in the AI Era (topic-2-new-skills-required-in-the-ai-era)

Students will investigate skills becoming more important because of AI-assisted software development.

Possible areas:
* prompt engineering,  
* critical thinking,  
* system design,  
* architecture understanding,  
* testing and verification,  
* communication and teamwork,  
* adaptability.

Main question:
* If AI can generate code faster, what skills become more valuable in software engineers?
